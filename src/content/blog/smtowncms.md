## 제작 기간
2019년 9월 9일 ~ 2020년 3월 2일<br/>
이 기간 동안, 프로젝트 팀은 사이트의 기획 및 설계 단계에서부터 개발, 테스트, 배포까지의 모든 과정을 체계적으로 진행했습니다. 
각 단계마다 피드백을 받고 수정하여 최종 결과물의 품질을 높였습니다.
기술 스택본 프로젝트에서는 다음과 같은 다양한 기술을 활용하여 전반적인 사이트를 구축하였습니다

## 기술 스텍
HTML: 웹 페이지의 구조를 정의하기 위해 사용되었습니다. 
콘텐츠의 구성 요소들을 효과적으로 배치하고, 접근성을 고려한 마크업을 구현하였습니다.

CSS & CSS3: 디자인과 스타일을 적용하기 위해 CSS 및 CSS3를 활용하였습니다. 
이를 통해 반응형 웹 디자인을 구현하고, 다양한 화면 크기에서 사용자 경험을 최적화했습니다. 
애니메이션과 트랜지션 효과를 통해 시각적으로 매력적인 요소를 추가했습니다.

JavaScript & jQuery: 페이지의 동적 요소와 상호작용을 처리하기 위해 JavaScript와 jQuery를 사용하였습니다. 
사용자 인터페이스를 개선하고, AJAX 요청을 통해 비동기적으로 데이터를 처리하여 더 나은 사용자 경험을 제공했습니다.
Bootstrap: 반응형 웹 디자인을 쉽게 구현하기 위해 Bootstrap 프레임워크를 사용하였습니다. 
이를 통해 빠른 개발과 일관된 디자인을 유지할 수 있었으며, 모바일 친화적인 UI를 설계하는 데 큰 도움이 되었습니다.
이러한 기술들을 종합적으로 활용하여 SMTOWN CMS 페이지를 구축함으로써, 사용자 친화적이며 시각적으로도 뛰어난 웹사이트를 완성했습니다.
